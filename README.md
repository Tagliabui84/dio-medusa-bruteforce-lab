# 🔐 Medusa Brute Force Lab

Laboratório de força bruta com Kali Linux e Medusa - Projeto DIO Santander Cybersecurity

## 📋 Sobre o Projeto

Este projeto documenta a implementação de um laboratório controlado para simulação de ataques de força bruta utilizando **Kali Linux** como máquina atacante e **Metasploitable 2** como alvo vulnerável.

### Cenários Implementados:
- ✅ Ataque de força bruta em **FTP**
- ✅ Ataque a formulário web (**DVWA**)
- ✅ **Password spraying** em SMB com enumeração de usuários

## 🖥️ Configuração do Ambiente

### Pré-requisitos
- VirtualBox
- Kali Linux (VM)
- Metasploitable 2 (VM)

### Configuração de Rede
- Configure ambas as VMs com **Host-Only Adapter**
- Verifique conectividade: `ping 192.168.56.x`
